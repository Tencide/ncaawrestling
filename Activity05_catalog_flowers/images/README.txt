Activity05 - Flower images folder

1. Download the flower images and description text files from Canvas (provided by professor).
2. Copy them into this folder (./images) and keep the directory structure from Canvas.
3. Optional: Download one extra flower image from https://www.pexels.com/search/free%20download/
   and add its description. You can reduce image size with https://tinypng.com/ or https://tinyjpg.com/
4. In comozeje_Activity05_catalog_flowers_bootstrap.html, update each card:
   - Set the <img src="images/yourfilename.jpg"> to match your image filenames.
   - Replace "Flower 1 Name", "Flower 2 Name", etc. with the actual flower names (keep <strong> for highlight).
   - Replace the paragraph text with the description from the text files for each flower.

Final zip: netid_Activity05_catalog_flowers_bootstrap.zip containing:
  - netId_Activity05_catalog_flowers_bootstrap.html
  - images/ (this folder with all flower images and text files)
