There are three ways to apply CSS to a HTML file. They are as follows:

  • 1. External stylesheet:
    • The External Stylesheet is a separated file with extension .css and is linked to the actual HTML file.
  • 2. Internal stylesheet:
    • The CSS Internal Stylesheet is a section of code within an HTML document. It is not necessary write an external file .css and it is not required add a link into the section <head>...</head>.
  • 3. Inline styles:
    • The CSS Inline Style is used to apply a unique style to a single HTML element.

1) External Stylesheet

The External Stylesheet is a separated file with extension .css and is linked to the actual HTML file.

The content of the .css file make references to the HTML via tags, classes or ids then the appearance of the Web page is improve it.

The advantages to use an external stylesheet document per each HTML document are the next :

  • The same .css file can be linked to different HTML Web pages if necessary.
  • It is easy to maintain your html page.
  • Change and Look process is simplified.
  • HTML Web pages are shorter. Then .css file is cached on the browser side, so the Internet traffic is reduced.

We must include a link in our HTML code. The link to the .css External Stylesheet is as follows : 


                <link rel="stylesheet" href="namefile.css" >

Where namefile.css is the name of the External Stylesheet. If the file is in different path or folder from the actual HTML file we must include it in the file name.

Start writing a HTML file named netid_Activity03_external.html such as the next code :


                <!DOCTYPE html>

2) Internal Stylesheet

The CSS Internal Stylesheet is a section of code within an HTML document. It is not necessary write an external file .css and it is not required add a link into the section <head>...</head>.

Let's write a HTML code including a CSS Internal Stylesheet in netid_Activity03_internal.html file.

The next code is a CSS Internal Stylesheet embedded in the <head>...</head> section.


                <!DOCTYPE html>
                <html>
                <head>
                   <style>

                      h2 {
                        color: whitesmoke;
                        background-color: brown;
                        font-family: cursive
                      }

                   </style>
                </head>
                <body>
                    <h3>2) Internal Stylesheet</h3>
                    <hr>

                    <h2>This is one example to show CSS Internal stylesheet. </h2>

                </body>
                </html>

Observe that the .css code is written directly inside the HTML document for improving the text of a tag h2.

The Internal Stylesheet has the next inconvenients :

  • If the .css code is extense, all the code will increase the size of the HTML document and its maintenance will become complex.
  • The .css is useful only in the actual document. If we need the same work in different HTML document, all .css has to be copied to the new HTML.

Add the text between tags <h2> ... </h2> and the CSS Internal stylesheet to the netid_Activity03_internal.html file.

3) Inline Style

The CSS Inline Style is used to apply a unique style to a single HTML element.

The following example sets the text color of the <h1> element to blue, and the text color of the <p> element to red:


                <h1 style="color:Green; background-color: transparent;">A Green Heading</h1>

                <p style="color:purple;">A purple paragraph.</p>

The style="color:Green;" will modify the appearance of the <h1> element in the HTML document.

And the style="color:purple;" will modify the appearance of the <p> element.

Add the following HTML of h1 and p to netid_Activity03_inline.html :


                <!DOCTYPE html>
                <html>
                <head>
                <title> Inline Stylesheet </title>
                </head>
                <body>
                    <h3>3) Inline Stylesheet</h3>
                    <hr>
                    <h1 style='color:Green; background-color: transparent;'>A Green Heading</h1>
                    <p style='color:purple;'>A purple paragraph.</p>
                </body>
                </html>

We can list several advantages of using inline styles. However, it is more important to highlight the disadvantages:

  • If the .css code is extensive, it increases the size of the HTML document and makes maintenance more complex.
  • The .css is applied only to the current document. If the same styling is needed in another HTML document, the entire .css code must be copied.

Summary :

CSS can be applied to HTML by the next three forms :

  • 1) External stylesheet
    by using a <link> element to link to an external CSS file.
  • 2) Internal stylesheet
    by using a <style>...</style> element in the <head>...</head> section.
  • 3) Inline styles
    by using the style attributes inside HTML elements.



CHALLENGE (Required)

Demonstrating External, Internal, and Inline CSS

The goal of this challenge is to demonstrate a clear understanding of the three different ways to apply CSS: external stylesheets, internal stylesheets, and inline styles. Each method must be implemented in a separate HTML file with intentional and visible design choices.

1) External Stylesheet Page

This page must demonstrate styling that comes entirely from an external CSS file. The external stylesheet should control the appearance of the entire page.

What to include:

  • A main container section using a div with a light gray background color and padding
  • One h1 heading with prominent styling (32px text size, orange color, center alignment)
  • One h2 subheading with different styling from the h1 (24px text size, dark blue color, left alignment)
  • At least one p paragraph with readable text styling (16px text size, dark gray color, left alignment)

2) Internal Stylesheet Page

This page must demonstrate styling using an internal stylesheet defined inside the HTML file. All visual styling should come from the internal style section.

What to include:

  • A main container section using a div with a light yellow background color and padding
  • One h1 heading with visible styling (30px text size, green color, center alignment)
  • One h3 subheading with different styling from the h1 (22px text size, dark green color, center alignment)
  • At least one p paragraph with readable text styling (15px text size, black color, justified or left alignment)

3) Inline Stylesheet Page

This page must demonstrate inline styling applied directly to HTML elements. Each styled element should clearly show visible design changes.

What to include:

  • A main container section using a div with a light blue background color and padding
  • One h1 heading with inline styling (28px text size, red color, center alignment)
  • One h2 or h3 subheading with different inline styling (20px text size, purple color, left alignment)
  • At least one p paragraph with inline styling (14px text size, dark blue color, left alignment)

General Requirements

  • Do not mix stylesheet types within the same HTML file
  • Each page must clearly demonstrate its assigned CSS method
  • Use clean, readable HTML structure
  • Follow file naming exactly as specified
  • Mixing multiple CSS styling methods within the same HTML file will result in a grading penalty.
