CSS: Cascading Style Sheets

CSS is a language to describe the presentation of a document written in HTML.

CSS is used to style and layout web pages. A web page is improved altering fonts, colors, size and space of content, columns animations and decorative elements.

CSS selectors :

  • Tag: A tag selector is sometimes referred to as a type name selector or element selector because it selects an HTML tag/element in your document.
  • Class: The class selector selects elements with a specific class attribute.
  • Id: The id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used

3) Id selector

The id selector in the <style>...</style> section begins with a # symbol rather than a dot (.) character for a class or the HTML tag. The id selector is used in the same way as we use a class selector. The class can be re-used in several parts of the HTML code; however, an id can be used only once per page. That is, the id selector can only be applied to the element pertaining this id.

This time, the next ids :

  • head3improve_1 assigned to header Aquaman,
  • head3improve_2 assigned to header Ant-Man, and
  • stanlee assigned to the name of the Author : Stan Lee

will improve the appearance of the previous page.

Instructions:

Write the next code in a NetId_moviespertid.html and add the properties to the CSS <style>...</style> section to change the appearance of the web page :

                text-align: left;
                color: #4CAF50;

            text-align: justify;
            letter-spacing: 3px;

                background-color: #4caf4f2a;
            }
            .highlightbook {
                font-weight: bold;
                background-color: #1d1f1dab;
                color: white;
            }
            .highlightyear{
                color: red;
            }
            #head3improve_1{
                color: black;
                font-size: xx-large;
                letter-spacing: 5px;
            }
            #head3improve_2{
                color: black;

    </style>
</head>

Created by <span id="stanlee">Stan Lee</span>, Larry Lieber and Jack Kirby, Ant-Man's first appearance was in Tales to Astonish #27 (<span class="highlightyear"> January 1962</span>) but first appeared in costume in Tales to Astonish #35 (<span class="highlightyear">September 1962</span>).
        </p>
    </div>

</body>
</html>
