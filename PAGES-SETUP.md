# GitHub Pages setup (one-time)

1. Open **https://github.com/Tencide/NCAA-WRESTLING-GAME**
2. Click **Settings**
3. In the left sidebar, under **Code and automation**, click **Pages**
4. Under **Build and deployment**, next to **Source**, choose **GitHub Actions** (not "Deploy from a branch")
5. Save. Then go to **Actions** and run **Deploy to GitHub Pages** once if needed.

Site URL after a successful deploy: **https://tencide.github.io/NCAA-WRESTLING-GAME/**
